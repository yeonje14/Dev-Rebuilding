## 🟢 PHASE 2 — 자료구조 & 알고리즘 집중 공략
**기간:** 3 – 5개월차
**언어:** ⚙️ C 심화 / 🐍 Python 병행 / 🌲 DS&A 핵심 구간

> 이 단계가 CS의 심장입니다.
> 자료구조와 알고리즘을 **C로 먼저 구현**해서 원리를 체감하고,
> 같은 구조를 Python으로 다시 짜며 두 언어의 차이를 비교합니다.

### 🌲 자료구조 — 선형
- [ ] 배열 (Array) — C로 직접 구현
- [ ] 연결리스트 (Linked List) — 단일 C로 구현
- [ ] 연결리스트 — 이중 C로 구현
- [ ] 스택 (Stack) — C로 구현 → Python으로 재구현
- [ ] 큐 (Queue) — C로 구현 → Python으로 재구현
- [ ] 덱 (Deque)
- [ ] **비교 정리:** 각 구조의 시간복잡도 표 작성

### 🌲 자료구조 — 비선형
- [ ] 이진트리 (Binary Tree) — 순회 3가지 (전위·중위·후위)
- [ ] 이진탐색트리 (BST) — 삽입 · 탐색 · 삭제
- [ ] 힙 (Heap) — 최대힙 · 최소힙
- [ ] 해시테이블 — 해시함수 + 충돌 해결 (체이닝)
- [ ] 그래프 — 인접행렬 vs 인접리스트 표현
- [ ] **비교 정리:** 언제 어떤 자료구조를 쓰는가

### ⚡ 알고리즘 — 기초
- [ ] 버블 정렬 · 선택 정렬 · 삽입 정렬
- [ ] 합병 정렬 (분할정복 이해)
- [ ] 퀵 정렬
- [ ] 이진 탐색
- [ ] 재귀 (팩토리얼 · 피보나치 · 하노이탑)
- [ ] 빅오 표기법 완전 정리 (O(1) ~ O(n²))

### ⚡ 알고리즘 — 그래프
- [ ] BFS (너비 우선 탐색) — 큐 활용
- [ ] DFS (깊이 우선 탐색) — 스택/재귀 활용
- [ ] 다익스트라 알고리즘 개념 이해
- [ ] BFS vs DFS 언제 쓰는가 정리

### ⚙️ C 심화
- [ ] 동적 메모리 할당 (malloc / free)
- [ ] 포인터로 연결리스트 직접 구현
- [ ] 메모리 누수 디버깅 경험
- [ ] 이중 포인터 이해

### 🗄️ 데이터베이스 기초
- [ ] 관계형 DB 개념 (테이블 · 키 · 관계)
- [ ] SQL 기초 — SELECT · WHERE · ORDER BY
- [ ] SQL 중급 — JOIN · GROUP BY · 집계함수
- [ ] SQLite로 미니 프로젝트 (간단한 CRUD)

### 🌐 네트워크 기초
- [ ] HTTP 요청/응답 사이클 이해
- [ ] TCP/IP 4계층 개념
- [ ] DNS · IP · 포트 개념
- [ ] REST API가 뭔지 설명할 수 있는 수준

### 🏆 LeetCode 훈련
- [ ] Easy 10문제 (C로 풀기)
- [ ] Easy 20문제 (Python으로 풀기)
- [ ] Easy 20문제 (복습 + 시간복잡도 분석)
- [ ] **목표: Easy 50문제 완료**

**🤖 AI 활용 예시**
```
"스택이 왜 필요한지 실생활 예시로 먼저 설명해줘."
→ "그럼 C로 스택 구현해줘."
→ 직접 작성
→ "같은 걸 Python list로 구현하면 어떻게 달라?"
→ "내가 설명해볼게, 틀린 부분 잡아줘."
```
